## Key Patterns

- Zustand store reset in tests: `useStore.setState(initialState)` + `localStorage.clear()` in `beforeEach`
- PWA hydration gate: check `useStore.persist.hasHydrated()` before rendering state-dependent UI
- Lives colors: `bg-green-500` (>3), `bg-yellow-400` (=3), `bg-red-500` (<3), `bg-gray-400` (dead)
- Card grid layout: `grid-cols-5` (9 values + "?"), `grid-cols-4` (4 manilha suits)
- Badge overlays: `relative` on button + `absolute -top-1 -right-1` on `<span>`
- Never use dynamic Tailwind class strings (e.g. `` `bg-${color}-500` ``) — they get purged
